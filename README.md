# GitCenter

Rust(Tauri)製デスクトップ向けGithubクライアント（になる予定）

Tauri + React + Typescript

## （とりあえず実装したい）機能

- Issues
  - これを作ろうと思った理由、むしろこれとリポジトリ操作以外実装しないかもしれない
  - Issueを見る・立てる・コメントするなど
 
- Github APIでできる範囲の機能
